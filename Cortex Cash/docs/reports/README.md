## Padrão de Nomenclatura
- Arquivos datados: `YYYY-MM-DD_tema-relatorio.md` (ex.: `2025-11-06_code-scan.md`)
- Relatórios por versão: `MODULE_REPORTS_V{MAJOR.MINOR}.md` (ex.: `FINANCE_REPORTS_V0.5.md`)
- Issues/Audit: `{AREA}_ISSUES_{TOPICO}.md` (ex.: `ARCHITECTURE_ISSUES_AI_USAGE.md`)
