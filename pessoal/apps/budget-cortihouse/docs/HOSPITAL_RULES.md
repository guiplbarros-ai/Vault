# Regras de Cortinas Hospitalares

## 1. COMPOSIÇÃO DA CORTINA

### Materiais

| Material | Especificação |
|----------|---------------|
| **Vinil** | VNS 45, antichama, antibacteriano, antifúngico, 0.30mm, dupla face lisa |
| **Tela Colméia** | 100% poliéster |
| **Trilho** | Suíço Luxo, alumínio, pintura eletrostática branca |
| **Ilhoses** | Niquelados |
| **Ganchos** | PVC/Nylon, espaçados a cada 15cm |
| **Deslizantes** | Para trilho |

### Cores Disponíveis
- Azul (padrão)
- Outras: verificar disponibilidade

---

## 2. MEDIDAS

### Estrutura da Cortina

```
┌───────────────────────────────┐
│         TELA COLMÉIA          │ ← 0.60m ou 0.90m
│     (parte superior, vazada)  │
├───────────────────────────────┤
│                               │
│                               │
│           VINIL               │ ← 2.00m (fixo)
│     (parte principal)         │
│                               │
│                               │
└───────────────────────────────┘
```

### Regra do Pé Direito

| Pé Direito | Tela Colméia | Total | Rebaixamento |
|------------|--------------|-------|--------------|
| ≤ 2.60m | 0.60m | 2.60m | Não |
| 2.61m - 3.10m | 0.90m | 2.90m | Não |
| > 3.10m | 0.90m | 2.90m | **SIM** |

### Rebaixamento

Quando pé direito > 3.10m, usa-se perfil de alumínio para baixar o ponto de fixação.

**Materiais adicionais:**
- Perfil de alumínio (metros)
- Suportes
- Parafusos
- Buchas
- Mão de obra extra

---

## 3. FATOR DE FRANZIDO

```
FATOR = 1.65 (65% a mais que a largura do vão)
```

**Exemplo:**
- Vão: 2.00m
- Largura do tecido: 2.00m × 1.65 = 3.30m

---

## 4. TRILHO

### Modelo: Suíço Luxo
- Material: Alumínio
- Acabamento: Pintura eletrostática branca
- Fixação: Direta no teto com suportes

### Curvas
- Preço: **R$ 30,00** por curva
- Uso: Quando trilho precisa contornar obstáculos

### Suportes
- Quantidade: ~1 a cada metro + 1
- Até 1.5m: 2 suportes
- 1.5m - 3.0m: 3 suportes
- Acima: calcular

## 7. EXEMPLO DE CÁLCULO

**Entrada:**
- Largura: 2.10m
- Pé direito: 2.60m
- Com trilho e instalação

**Cálculo:**

```
1. Largura do tecido = 2.10 × 1.65 = 3.465m

2. Vinil:
   - Área = 3.465m × 2.00m = 6.93m²

3. Tela Colméia (pé direito ≤ 2.60, usa 0.60m):
   - Área = 3.465m × 0.60m = 2.08m²

4. Acessórios (a cada 15cm):
   - Ganchos = ceil(3.465 / 0.15) = 24 un
   - Ilhoses = 24 un
   - Deslizantes = 24 un

5. Trilho:
   - Comprimento = 2.10m
   - Suportes = 3 un
   - Tampas = 1 par

6. VALORES (com preços exemplo):
   - Vinil: 6.93m² × R$ 85 = R$ 589,05
   - Tela: 2.08m² × R$ 45 = R$ 93,60
   - Trilho: 2.10m × R$ 120 = R$ 252,00
   - Suportes: 3 × R$ 25 = R$ 75,00
   - Tampas: 1 × R$ 30 = R$ 30,00
   - Acessórios: ~R$ 50,00
   - Mão de obra: R$ 200,00
   ─────────────────────────────
   TOTAL: ~R$ 1.289,65
```
